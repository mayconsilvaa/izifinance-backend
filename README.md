## Como fazer o download e executar o projeto

```bash

  # Clonar o diretório
  $ git clone https://github.com/mayconsilvaa/izifinance-backend.git

  # Acessar o diretório
  $ cd izifinance-backend

  # Instalar as dependêncidas
  $ yarn

  # Executar o projeto
  $ yarn dev

```
